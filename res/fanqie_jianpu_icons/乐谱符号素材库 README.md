# 乐谱符号素材库 README

## 一、项目简介

本素材包含 `1.png` \~ `52.png` 共 52 个乐理符号图片，适配简谱 / 五线谱打谱、乐谱 OCR 识别、制谱软件开发；配套预览图与映射文本文件。
配套文件：

- `symbol.gif`：全部符号预览总图

- `symbol_1.txt ~ symbol_6.txt`：符号名称、图片映射配置表
